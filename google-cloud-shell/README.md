# 1단계: 프로젝트 생성 및 API 활성화 (A, B 계정 각각 수행)
먼저 SA들이 활동할 '건물(프로젝트)'을 지어야 합니다.

구글 클라우드 콘솔 접속: https://console.cloud.google.com/

프로젝트 만들기:

상단 로고 옆의 프로젝트 선택 창 클릭 -> [새 프로젝트]

이름 예시: migration-project-a (B계정은 migration-project-b)

만들기 클릭 후 잠시 대기 -> 알림창에서 [프로젝트 선택] 눌러서 해당 프로젝트로 들어갑니다.

API 활성화:

좌측 메뉴(햄버거) -> [API 및 서비스] -> [라이브러리]

검색창에 Google Drive API 검색 -> 클릭 -> [사용(Enable)] 버튼 클릭.

2단계: SA 100개 자동 생성 및 키 다운로드 (A, B 계정 각각 수행)
이제 명령어를 입력해서 로봇 100마리를 생산합니다. 별도 프로그램 설치 없이 웹에서 바로 됩니다.

1. Cloud Shell 실행:

화면 우측 상단에 있는 **[ >_ ] 모양 아이콘 (Cloud Shell 활성화)**을 클릭합니다.

화면 아래에 검은색 터미널 창이 열립니다.

2. 명령어 붙여넣기 (한 번에 복사해서 붙여넣으세요):

아래 스크립트는 SA 100개를 만들고, 키(JSON)를 생성하고, 이메일 리스트를 뽑아줍니다.

(약 5~10분 정도 걸릴 수 있습니다)

3. 파일 내 컴퓨터로 다운로드:

터미널 창 상단에 [더보기(⋮)] 버튼 클릭 -> [파일 다운로드]

파일 경로에 sa_keys.zip 입력 -> [다운로드]

[중요] sa_keys/email_list.txt 파일도 따로 열어서 내용을 복사해두거나 다운로드하세요. (그룹스 추가용)

👉 결과: A 계정에서 zip 파일 하나, B 계정에서 zip 파일 하나, 총 200개의 json 키 파일과 이메일 리스트를 확보하게 됩니다.
